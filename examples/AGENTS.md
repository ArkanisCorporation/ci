# examples/AGENTS.md

## Scope

Applies consumer example repos under `examples/**`.

## Rules

- Examples show minimal consumer wrapper, not platform internals.
- Use full-SHA placeholders: `<full-sha>`.
- `permissions: {}` at top.
- Job permissions minimal.
- No real secrets, org names, package IDs, cluster names.
- Each example must be copyable after replacing placeholders.
- Examples must include expected artifacts/outputs in comments or README.
- Keep examples small; docs explain details.
